Colymn

A lightweight, pure javascript grid system

Features

Leightweight

less than 1kb after compression

Responsive

with upto 7 breakpoints

Compaitible

Works in IE8

Symantic

Write cleaner html! No need to add clearfix containers to content

Simple

less than 1kb after compression

features

set up

1. Include colymn.min.js in the <head>

<script src="js/colymn.min.js"></script>

2. Call the plugin before the closing <body> tag

  <script>
    colymn();
  </script>
</body>

simple!

How to use

There are 3 parts: the prefix, the type and the fraction.

The prefix: 'xx', 'pp', 'pl', 'tp', 'tl', 'dt' and 'hd'. These stand for all-sizes, phone-portrait, phone-landscape, tablet-portrait, table-landscape, desktop and full hd monitors

Add classes like dt-col-3-4

Parallel columns

<div class="dt-col-3-4"></div>
<div class="dt-col-1-4"></div>

Centered columns

<div class="dt-center-31-44"></div>

offset columns

<div class="dt-col-4-9"></div>
<div class="dt-col-3-9 dt-offset-2-9"></div>

displaced columns

<div class="dt-center-5-6">
  <div class="dt-col-5-8 dt-left-1-12"></div>
</div>
