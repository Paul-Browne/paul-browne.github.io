A lightweight, pure javascript grid system

Features

Leightweight

1kb after compression

Responsive

with upto 7 breakpoints

Compaitible

Works in IE9

Symantic

Write cleaner html! No need to add clearfix containers to content

Simple

If you understand fractions, you be fine

 

set up

1. Include colymn.min.js in the <head>

<script src="js/colymn.min.js"></script>

2. Call the plugin before the closing <body> tag

  <script>
    colymn();
  </script>
</body>

or call with options

  <script>
    colymn({
      margin: 4,
      phonePortrait: 300,
      phoneLandscape: 300,
      tabletPortrait: 440,
      tabletLandscape: 640,
      desktop: 1024,
      fullHD: 1440
    });
  </script>
</body>

How to use

There are 3 parts: the prefix, the type and the amount.

The prefix:

prefix device size
xx-

all devices

greater than 0 pixels wide
pp-

phone portrait

greater than 300 pixels wide and in portrait
pl-

phone landscape

greater than 300 pixels wide and in landscape
tp-

tablet landscape

greater than 440 pixels wide and in portrait
tl-

tablet landscape

greater than 640 pixels wide and in landscape
dt-

desktop

greater than 1024 pixels wide
hd-

Full HD

greater than 1440 pixels wide

The type:

type device rules
col-

column

center-

1 column, centered

offset-

offset a column to the right

must be accompanied with a col- type
left-

push/displace a column to the left

must be accompanied with a col- type
right-

push/displace a column to the right

must be accompanied with a col- type
type device rules
prefix-col-x-y

defines column of width x from a total of y
prefix-center-x-y

defines one centered column of width x from a total of y

prefix-center-x(px|em)

defines one centered column of a fixed width x, in either pixels or ems

prefix-offset-x-y

offsets a given column (not centered columns)

must be accompanied with a col- type
prefix-left-X-Y

push/displace a column to the left

must be accompanied with a col- type
prefix-right-X-Y

push/displace a column to the right

must be accompanied with a col- type
