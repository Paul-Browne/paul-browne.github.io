Grid System

overview

epicGrid controls the layout. With epicGrid there are two types of columns, and two types of "additions" which add other style to the columns

Column types Description
prefix-col-x-y A column for the chosen viewport. X number of columns wide from a total of Y
prefix-center-x-y Center the column for that viewport. X number of columns wide from a total of Y
prefix-offset-x-y Offset the column to the right by X number of columns from a total of Y
prefix-left-x-y Displace a column to the left by X number of columns from a total of Y
prefix-right-x-y Displace a column to the right by X number of columns from a total of Y
  • choose one of the prefixes x, s, m, l or h for that viewport
  • classes left, right and offset must be accompanied with a col class to specify width.
  • the class center cannot be used with left, right or offset, only col.
breakpoints

You can use up to 5 different prefixes to independently control the layout for different sizes

size prefix breakpoint
eXtra small x- greater than 0
Ssmall s- greater than 320px
Medium m- greater than 656px
Large l- greater than 1024px
Huge h- greater than 1440px

Image Tools

responsive images

Responsive images - images that change depending on container size and device pixel density, are easy to add. The example below shows 5 different images that will be loaded depending on the afore mentioned conditions. The placeholder image is loaded regardless, but is entierly optional. You can specify as few, or as many breakpoints as you like.

<img src="i/placehoder.jpg" class="foo" id="bar" alt=""
data-src="<320:i/small.jpg,
      <600:i/medium.jpg,
      <1024:i/large.jpg,
      <1440:i/huge.jpg,
      >1440:i/massive.jpg" />
image cover

Adding the class icovr to a container of an <img> will gaurentee that the image covers the container and is centered. In the same way that background-size: cover; background-position: center center; works for background images. A height for the container must be specified in order for it to work properly.

<div class="icovr" id="bar">
    <img src="i/sunset.jpg" alt="" />
    </div>
